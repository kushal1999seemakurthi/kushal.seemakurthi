# 🛡️ Aegis: Agentic Auto-Healing Test Loop

Aegis is an automated CI/CD pipeline that pairs Playwright's visual and functional testing with an agentic LLM workflow. When a regression is detected, Aegis doesn't just fail the build—it captures the context, spins up a coding agent, and attempts to autonomously patch the codebase.

## ⚙️ Architecture Overview

The loop operates in four distinct phases:
1. **Detection:** Playwright runs headless tests (DOM assertions, Visual Regression via Percy, and WebGL Canvas snapshots).
2. **Context Aggregation:** On failure, a Node script captures the stack trace, network logs, DOM state, and Git diff.
3. **Agentic Resolution:** The payload is piped into an agentic CLI tool powered by the Gemini API.
4. **Validation:** The agent applies a patch to a detached branch and triggers a dry-run test. If successful, it opens an Auto-PR.

## 🛠️ Prerequisites

* Node.js v20+
* Playwright (`npm init playwright@latest`)
* An agentic CLI tool (e.g., Aider or a custom Python wrapper)
* Active API Keys exported in your environment:
  * `GEMINI_API_KEY` (For the reasoning and code generation engine)
  * `GITHUB_TOKEN` (For opening Auto-PRs)
